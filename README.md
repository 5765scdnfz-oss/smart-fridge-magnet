# 智能冰箱贴 🧊

> 黑客松项目 · 2026-08-01 展示

## 项目简介

通过AI对话 + 拍照识别，解决"每天不知道吃什么"的痛点。

## 核心功能

1. **拍照识别** — 拍冰箱照片，自动识别食材种类、数量、保质期
2. **智能推荐** — 根据家庭画像 + 库存 + 时间，推荐A/B菜谱方案
3. **库存管理** — 确认菜谱后自动扣减库存

## 技术栈

- **后端：** Python + Flask
- **数据库：** SQLite
- **AI模型：** MiMo v2.5（图片识别）+ MiMo v2.5-pro（对话推理）
- **定时任务：** APScheduler

## 我负责的部分

```
拍照 → 识别食材 → 存入库存
        ↓
家庭画像 + 库存 + 时间 → 推荐菜谱（A/B方案）
        ↓
确认方案 → 扣减库存
```
